# 🏥 SGHSS - Sistema de Gestão Hospitalar e de Serviços de Saúde (VidaPlus)

Projeto desenvolvido como atividade acadêmica do curso de **Análise e Desenvolvimento de Sistemas – UNINTER**.

O sistema SGHSS (VidaPlus) consiste em uma API RESTful para gerenciamento hospitalar, permitindo autenticação de usuários e cadastro de pacientes de forma segura.

---

## 🚀 Tecnologias Utilizadas

- Python
- Flask
- SQLAlchemy
- JWT Authentication
- SQLite
- REST API
- Git & GitHub

---

## 📌 Funcionalidades

✅ Autenticação de usuários  
✅ Geração de Token JWT  
✅ Cadastro de pacientes  
✅ API RESTful  
✅ Estrutura backend em Python  

---

## 🔐 Autenticação

O sistema utiliza autenticação baseada em **JSON Web Token (JWT)** garantindo acesso seguro às rotas protegidas.
